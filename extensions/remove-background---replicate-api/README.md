# Remove Background

Remove image backgrounds instantly using AI.

## Setup

**Get your Replicate API token:**

1. Sign up at [replicate.com](https://replicate.com).
2. Go to [API tokens](https://replicate.com/account/api-tokens) and create a token.
3. Copy the token (starts with `r8_`).

**Configure in Raycast:**

- Open Extensions â†’ Remove Background.
- Add your API token.

## How to Use

1. Select a PNG or JPG image in Finder (macOS) or use the file picker.
2. The extension processes it automatically.
3. View results side-by-side and copy/paste as needed.

## Actions

- Copy processed image (paste anywhere).
- Copy image URL.
- Paste image URL.
- Open in browser.

## Requirements

- Internet connection.
- Replicate API token.

## Troubleshooting

- **No image?** Select one in Finder or use picker.
- **Errors?** Check token and connection.
- **Formats?** PNG/JPG only.

Need help? See [Replicate docs](https://replicate.com/docs).
